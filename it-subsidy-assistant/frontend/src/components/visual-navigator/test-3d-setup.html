🎯 Visual AI Navigator - 3D補助金探索システム

概要

Visual AI Navigatorは、IT補助金の選択プロセスを革新的な3D空間で可視化するシステムです。 企業のプロファイルに基づいて、AIが最適な補助金への経路を自動的に計算し、 インタラクティブな3D環境で表示します。

実装済み機能 Ready

🌐 3Dビジュアライゼーション

  • 補助金ノードの3D配置
  • インタラクティブなカメラコントロール
  • ホバー・選択エフェクト
  • パーティクルエフェクト

🤖 AIパス計算

  • 企業プロファイル分析
  • マッチングスコア計算
  • 最適経路の自動生成
  • 信頼度スコア表示

✨ ビジュアルエフェクト

  • Bloomポストプロセッシング
  • 被写界深度エフェクト
  • アニメーションパス
  • グローエフェクト

📊 AI推奨パネル

  • リアルタイム分析結果
  • 推奨理由の説明
  • 申請順序の提案
  • 適合度分析

技術スタック

React TypeScript Three.js React Three Fiber @react-three/drei @react-three/postprocessing Framer Motion Leva (Debug UI)

アクセス方法

  1. フロントエンドサーバーを起動: npm run dev
  2. ブラウザで以下のURLにアクセス:
    • 直接アクセス: http://localhost:5173/visual-navigator
    • 革新的機能デモから: http://localhost:5173/innovative-demo
    • ヘッダーメニューから: 「🎯 3Dナビゲーター」をクリック

操作方法

  • マウスドラッグ: 3D空間の回転
  • マウスホイール: ズームイン/アウト
  • ノードクリック: 補助金の詳細表示とフォーカス
  • コントロールバー: カメラリセット、フルスクリーン切り替えなど
  • AI推奨パネル: 左側のパネルで推奨事項を確認
